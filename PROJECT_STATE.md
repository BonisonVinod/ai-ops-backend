# Agentic Mind - Master State & Logic

## CORE MISSION
A platform to analyze SOPs/JDs for "Digital Co-worker" augmentation. Focus on ROI, Precision, and Human-Supervised automation.

## LOGIC PARAMETERS
1. **Precision Assessment:** Analyze tasks for "Error Sensitivity." (e.g., Financial tasks = High Precision).
2. **Approval Workflow:** By default, the Human stays in the loop for ALL final outputs. Automation only handles the *execution* of the task.
3. **Skill Mapping:** Extract the exact skills required to fulfill the JD/SOP.
4. **Pricing (India Market):**
   - JD: 20% of Annual CTC for setup.
   - SOP: ₹25k - ₹75k based on logic steps.
   - Subscription: Tiered (Standard/Pro/Enterprise).
